# Logo / İkon Dosyaları

Bu klasöre kendi logonuzu ekleyin:

| Dosya | Kullanım | Önerilen boyut |
|-------|----------|----------------|
| `icon.png` | Pencere ikonu (başlık çubuğu, görev çubuğu) | 32x32 veya 64x64 px |
| `icon.ico` | Windows .exe ikonu (derleme için) | 256x256 px |
| `icon.icns` | macOS .app ikonu (derleme için) | 512x512 px |

**icon.png** mutlaka olmalı – uygulama çalışırken pencere ikonu için kullanılır.

**icon.ico** ve **icon.icns** isteğe bağlı – sadece `python build.py` ile derleme yaparken exe/app ikonu için kullanılır.
